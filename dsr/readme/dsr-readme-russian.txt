Ритуал открытия Нечестивых Врат (для BG2 - Трон Баала)

Содержание: Мод создает Книгу ритуала открытия Нечестивых Врат, которая находится у волшебника Толгериаса.
Если вам удастся ее отнять, то вы сможете раз в день вызывать различных демонов случайным образом.

Внимание: Для соблюдения баланса Толгериас сделан более сильным соперником :-)

Кстати, этой книгой не могут пользоваться персонажи с добрым мировоззрением.

Наслаждайтесь, этот мод сделан для вас!

TIN (переделка в WeiDU-формат - Badgert)
